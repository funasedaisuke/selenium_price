Overview

## Description
- ネットスーパーから商品名、価格を取得するアプリ

## Requirement
- git
- docker

## Usage

```
git clone https://github.com/uh-zz/selenium_price.git

cd selenium_price

docker-compose up -d

docker exec -it python /bin/bash

python ito-yokado.py
```
